# WavTool
Tiny single-file C++17 WAV utility (16-bit PCM, mono/stereo).

## Features
- `info` — print metadata
- `gain <dB>` — apply gain
- `normalize` — peak-normalize to −1 dBFS
- `fade <in_ms> <out_ms>` — linear fades
- `lowpass <cutoff_hz> [Q]` — simple biquad LPF (Q default 0.707)
- `synth-sine <freq> <sec> [sr] [amp]` — generate a sine

## Build
```bash
g++ -std=gnu++17 -O2 -o wavtool wavtool.cpp


USAGE
./wavtool info input.wav
./wavtool gain -6 input.wav output.wav
./wavtool normalize input.wav output.wav
./wavtool fade 50 300 input.wav output.wav
./wavtool lowpass 1200 0.7 input.wav output.wav
./wavtool synth-sine 440 2.5 48000 0.5 sine.wav
